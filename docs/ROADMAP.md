# Roadmap

- Lexer/Parser
	- Aceite: AST gerado para exemplos básicos
- Type Checker
	- Aceite: erros de tipo simples e coerção mínima

- Esparsidade declarativa
	- Aceite: seleção automática de kernels sparsos

## 2028 — Maturidade

- Self-hosted
	- Aceite: bootstrap do compilador
- Pacotes oficiais
	- Aceite: runtime e libs iniciais
