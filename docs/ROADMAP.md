# Roadmap

## Objetivo

Resumo do plano de evolução do projeto.

## Curto prazo

- Consolidar SPEC v0.1 (ajustes finos e exemplos validados).
- Melhorar typechecker (constraints e diagnósticos).
- Codegen mínimo para `fn main()`.

## Médio prazo

- Formatter oficial.
- Linter mínimo.
- Language Server básico.

## Longo prazo

- FFI com C/Rust.
- ABI documentada.
- Benchmarks públicos.
