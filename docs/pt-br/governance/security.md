# Política de Segurança

## Propósito

Descrever o processo de divulgação responsável de vulnerabilidades.

## Relato

Reporte vulnerabilidades via [GitHub Issues](https://github.com/marciopaiva/tupalang/issues).

## Escopo

- CLI
- Parser/Verificador de tipos
- Codegen

## SLA

- Reconhecimento: em até 5 dias úteis
- Atualização inicial: em até 10 dias úteis
