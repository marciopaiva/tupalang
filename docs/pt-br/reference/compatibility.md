# Guia de Compatibilidade

## Propósito

Definir versões mínimas e suporte de plataforma.

## Toolchain

- Rust: stable recente (recomendado: stable mais recente).
- LLVM: necessário apenas para codegen futuro.

## Sistemas suportados

- Linux: suportado.
- macOS: suportado.
- Windows via WSL: suportado.

## Notas

- Codegen ainda é um stub; requisitos do LLVM serão detalhados quando o backend for implementado.
