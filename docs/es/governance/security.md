# Política de Seguridad

## Propósito

Describir el proceso de divulgación responsable de vulnerabilidades.

## Reporte

Reporta vulnerabilidades vía [GitHub Issues](https://github.com/marciopaiva/tupalang/issues).

## Alcance

- CLI
- Parser/Verificador de tipos
- Codegen

## SLA

- Acuse de recibo: dentro de 5 días hábiles
- Actualización inicial: dentro de 10 días hábiles
