# Glosario

## Propósito

Definir términos clave usados en el lenguaje y la documentación.

## Términos

- **Alineación**: conjunto de restricciones éticas verificadas en tiempo de compilación.
- **Restricción**: condición que debe probarse para un tipo `Safe<T, ...>` (por ejemplo `!nan`, `!hate_speech`).
- **Densidad**: parámetro de tipo que controla la esparsidad de tensores.
- **EBNF**: notación formal para gramática de lenguajes.
- **Nabla (`∇`)**: operador nativo de diferenciabilidad.
- **Tipo Safe**: tipo anotado con restricciones, por ejemplo `Safe<f64, !nan>` o `Safe<string, !misinformation>`.
- **Span**: intervalo de texto usado para señalar errores (línea/columna).
- **Verificador de tipos**: verificador de tipos estático del compilador.
