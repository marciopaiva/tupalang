# Guía de Compatibilidad

## Propósito

Definir versiones mínimas y soporte de plataforma.

## Toolchain

- Rust: stable reciente (recomendado: stable más reciente).
- LLVM: requerido solo para codegen futuro.

## Sistemas soportados

- Linux: soportado.
- macOS: soportado.
- Windows vía WSL: soportado.

## Notas

- Codegen todavía es un stub; los requisitos de LLVM se detallarán cuando el backend sea implementado.
